== Description ==
Implements a shortcode [hellwoshoco] which will output Hello World!

Ordered list:

1. Place [hellwoshoco] anywhere in your site
2. It will output Hello World!
3. That\'s it.

== Installation ==
1. Upload \\\\\"hellwoshoco.php\\\\\\\\" to the \\\"/wp-content/plugins/\\" directory.
2. Activate the plugin through the \\\\\"Plugins\\\\" menu in WordPress.
3. Place [hellwoshoco] anywhere you want.
